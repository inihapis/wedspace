# Release Notes — Wedspace v0.0.2

**Release Date**: May 1, 2026  
**Version**: v0.0.2  
**Status**: ✅ Ready for Production Deployment

---

## 🎉 What's New

Wedspace v0.0.2 adalah rilis yang fokus pada **mobile experience improvements**, **data sync reliability**, dan **brand positioning refinement**.

### Major Changes

#### 1. Mobile Experience Enhancements 📱
- **Admin Mobile Header** — Simplified header dengan direct logout button (no redundant menu)
- **Auto-refresh on Focus** — Data automatically refreshes when app comes to foreground
- **Private Browsing Detection** — Detects and warns users about data persistence limitations
- **Theme Color Fix** — Safari mobile address bar now displays correct dark color

#### 2. Brand Positioning Refinement 🎯
- **Updated Meta Description** — Changed from "Persiapan nikah jadi lebih tenang" to "Platform yang membantu Anda mempersiapkan pernikahan dengan lebih terorganisir"
- **AuthPage Tagline Update** — Reflects helper platform role, not wedding organizer
- **Honest Positioning** — Emphasizes that Wedspace helps organize, not manages everything

#### 3. Data Sync Reliability 🔄
- **Focus Event Listener** — Automatically loads fresh data when app regains focus
- **Private Browsing Handling** — Gracefully handles localStorage unavailability
- **Session Persistence** — Better token management and session validation
