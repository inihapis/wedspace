# Release Notes — Wedspace v0.0.1a

**Release Date**: May 2026  
**Version**: v0.0.1a  
**Status**: ✅ Ready for Staging Deployment

---

## 🎉 What's New

Wedspace v0.0.1a adalah rilis alpha yang fokus pada **component refactoring** dan **navigation polish** untuk meningkatkan maintainability dan user experience.

### Major Changes

#### 1. Component Refactoring ✨
- **Removed all inline styles** — Semua komponen sekarang menggunakan Tailwind classes
- **Created reusable components** — Card, Button, Input, Section, Badge untuk konsistensi UI
- **Organized component structure** — Folder terstruktur: layouts/, pages/, shared/, features/

#### 2. Navigation Improvements 🧭
- **Centralized navigation configuration** — Semua navigation items di `client/src/data/navigation.js`
- **Mobile center positioning** — `center: true` property untuk item yang ditampilkan di tengah
- **Admin navigation polish** — AdminBottomNav sekarang support center positioning

#### 3. Premium Charts Standardization 📊
- **Consistent Card component usage** — Semua premium chart menggunakan Card wrapper
- **LockedChartCard pattern** — Locked state menampilkan blurred preview dengan lock overlay
- **Chart type correction** — "Rencana vs Aktual" sekarang menggunakan BarChart (horizontal)

#### 4. Tailwind CSS Integration 🎨
- **CSS variables mapping** — `bg-primary`, `text-text`, `shadow-sm`, dll
- **Consistent styling** — Semua warna dan spacing menggunakan design tokens
- **Dynamic values support** — Inline styles hanya untuk gradients, opacity, filters
